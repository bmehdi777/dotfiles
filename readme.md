# Dotfiles

- dotfile manager : _chezmoi_
- shell : _zsh_
    - using _oh-my-zsh_
- editor : _neovim_
- terminal manager : _tmux_
- tools :
    - lazygit
    - git-delta (see/gitdiff side by side)

## Linux

- window manager : _bspwm_ and _sxhkd_
- terminal : _alacritty_ or _kitty_
- screenshot: _flameshot_

## Macos

- terminal : _alacritty_ or _kitty_
